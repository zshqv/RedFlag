Severity Scoring Logic

Each finding is scored 0–100 based on:

  • Base Score: 50
  • +20 if severity tier is HIGH (sentiment < -0.25)
  • -15 if mitigating language present (e.g., "we believe", "resolved")
  • +15 if amplifying language present (e.g., "material adverse", "cannot guarantee")
  • Final Score: Clamped to 0–100
